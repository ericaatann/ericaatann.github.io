Projects

CriminalIntent

Description: CriminalIntent is an Android app I developed as part of my coursework. It allows users to keep a record of "crimes" they encounter in their daily lives. The app includes features like date and time recording, crime classification, and the option to add notes.

Technologies: Android, Kotlin, Android Studio.

PhotoGallery

Description: I worked on the PhotoGallery app from the Bignerdranch Android Programming book. The app fetches and displays photos from Flickr, allowing users to browse and view images. I learned about APIs, background services, and advanced UI components in the process.

Technologies: Android, Kotlin, API, Android Studio.

Tic-Tac-Toe Game

Description: I created a simple web-based Tic-Tac-Toe game using HTML, CSS, JavaScript, and React. I also utilized Node.js. The game is super simple as it is only single player. It's a great example of my front-end development skills.

Technologies: HTML, CSS, JavaScript, React, Node.js.
